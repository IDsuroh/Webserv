Welcome!

If you can see this page, the basic GET handling and static file serving are working. Use the tiles below to navigate to different parts of our Webserv project.

Browse Files

Lists the contents of www/files/. Great for testing autoindex and directory handling.

Go to /files/

Upload Area

Endpoint backed by www/upload/. Use it to test POST request and file uploads.

Go to /upload/

Trash Area

Files under www/trash/ can be listed with GET and removed with DELETE requests.

Go to /trash/

CGI Test

Runs cgi-bin/test.py. Use this to check our CGI implementation for GET and POST.

Run CGI

Readonly Endpoint

A simple endpoint to trigger 405 Method Not Alowed for POST/DELETE while allowing GET.

Go to /stress/readonly

Stress Endpoints
